CityUHK EE Courses

January 4, 2026 · Shuo

关于 CityUHK EE Dept. 的课程相关评价，这些课程主要集中在 EE Dept. 的 MSc / MPhil / PhD 研究生项目

CityUHK EIE Programme

香港城市大学电机工程系的 EIE 项目（后改名 EEE 项目），整体就读体验为高强度，Due 遍布整个学期，Assignment，Test 和 Midterm 无缝衔接。想体验大学卷保研生活的可以考虑来，弥补了大学没有保研计划同学的卷卷生活

课程分布为 1 / 2 通信课程，1 / 4 软件课程，1 / 4 硬件课程，MSc 的课程和 PhD 一起上课

整体师资力量雄厚，IEEE Fellows 众多，电机工程系是城大的王牌专业，Prof 对城大和城大 MSc 项目的学生比较认可，新生见面会直言每年 PhD 招生会给 MSc 项目毕业生十个左右的名额，但估计比较吃本科背景

城大 AC1、AC3 的食物在港校里面比较美味，学校氛围可以，香港学生、大陆学生、国际学生相处都比较友好

EE5808 Topics in Computer Graphics

课程主要内容：C / OpenGL，图形学坐标系变换，投影，动画，阴影，材质，深度检测，辐射场，先进技术（光线追踪，高斯泼溅，NeRF 等）

分数分布：4 次作业 （4 x 1.5 分），2 次 Test（2 x 15 分），1 个 mini Project（15 分），Final （50 分）

通过难度：较难，高分难度：极难

这门课有 UG 和 PG 版本，课程代码都为 EE5808，UG 和 PG 的课程不在一起上课，课程标准不同

Kelvin 的课比较枯燥，平时去的人大约为 35 / 70，偶遇节假日加课出勤率为 13 / 70，两次 Test 均分为 57 和 34，期末考试题型与 Test 有相似重合。第一节课 Kelvin 直说这门课大部分人只能拿 B / B-，A 和 F 都会很少；实际身边统计学显示学的很认真的同学拿的 B，平时 Test 在平均分附近或略高于平均分的同学在 C- / C / C+

总的来说不是很推荐，给分较低，事情较多，除非有志于 Graphics 的同学，这门课确实可以简单了解一下图形学的框架，并且比本科课程更加深入，但是不涉及到 Shader 的具体细节

EE5438 Applied Deep Learning

课程主要内容：MLP，CNN，RNN，Transformer，LLM

分数分布：Assignments: 15%，Quiz: 5%，Midterm: 10%，Group Project: 20%，Final Exam: 50%

通过难度：简单，高分难度：比较简单

PO 的神课，课上有签到，授课质量极高，PO 有自己写 Medium 文章，课件量巨大，Medium 阅读材料钜细，每次上课基本能到 70% ~ 80% 的出席率，PO 上课有激情，口头禅是 OK。Quiz 和 Test 分数不会太低，Final 与 Test 内容形式相近，题量较大，每道题分值小，比较难写完，也不会出现完全不会的低分情况。作业为三部分：知识点概念，计算题，Colab 修改 / 微调模型；作业量适中

强烈推荐此课程，认真学习可以入门深度学习和 LLM，如果把课件和文章吃透可以面试 AI 非算法类的岗位

EE6610 Queue’g Ther with Telec’n Apns

课程主要内容：（概率论和数理统计，随机过程），排队理论和各种排队模型

分数分布：15% Midterm，35% Group Learning，50% Final

通过难度：简单，高分难度：适中

Moshe 的课，Moshe 年纪比较大了据说是退休返聘来上课的。上课比较喜欢互动，有时候会讲中文比较幽默，上课内容和课件内容比考试难很多，课程核心作业为自由安排量的 Group Learning，意思是在 Canvas 讨论区水贴发自己的学习笔记和结果，每周都要发一次或者几次。学习强度由个人决定，想把课件都看懂非常难，想通过考试和课程比较容易

这门课是通信领域的偏基础理论课程，想往通信领域发展的同学必选，考虑到给分和灵活的强度，零基础的同学也推荐

EE6615 Nan’ for Dev’ and Mic’

课程主要内容：晶圆清洁和晶体生长，光刻，蚀刻，掺杂，氧化，先进技术（DUV，EUV，相移掩膜等）

分数分布：10% Cleanroom Lab Report，15% Assignment，25% Test，50% Final

通过难度：比较简单，高分难度：较难

Stella 的课，课程难度个人体感不大，改卷子可能比较严格，戏称藤校标准。出题某些题目方向可能比较微妙，大家都不会做。课程内容在于理解，个人感觉理解了不算太难，身边也有同学表示难以理解课程内容。整个课程内容分布比较平坦，没有特别的侧重点，课程事情少，适合零基础

整体难度不大，零基础也可以拿较高分数，可以很快速地了解 Fab 的所有工艺流程

SYE6204 VULSI Proc’s Intergration

课程主要内容：MOS Transistors，工艺制造和集成

分数分布：35% Midterm，35% Final，35% Group Project

通过难度：比较简单，高分难度：较难

秦琦的课，海归老师第一年来城大，本科南京大学，博士麻省理工，第一节课就直说不会为难大家。课程主讲器件物理特性和一小部分制造工艺，其中制造工艺和 EE6615 纳米技术有高度重合，可两节课联动学习事半功倍。课程整体难度偏大，考试与作业题目相似，考试通过难度不大。

课程 PPT 内容对非微电子，非半导体物理专业的同学非常难，如果要弄懂概念就得花点时间，考试题和作业难度都非常难，要自己看到题有思路就会做极难，总的来说这门课上课内容很充实，考试允许使用 Cheetsheet，通过难度不大，比较推荐
