Vim 首行乱码问题解决方法

在使用 Vim 时遇到了首行乱码形如"+q436f+q6b75+q6b64"，错误的乱码“掩盖”了正确的内容，却不会覆盖正确的内容，但能够导致首行的内容无法被查看，解决这项问题的方法是在 ".vimrc" 文件中添加 "set t_RV=" 确保 Vim 能够显示正确的内容。
