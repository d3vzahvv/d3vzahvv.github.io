矢量分析初步

· 2021-05-26 · # 数学工具 # 随笔

矢量分析初步主要介绍 "梯度, 散度, 旋度" 的概念及其之间的关系

标量场和矢量场

标量场

标量场是物理量为标量所确定的场, 如温度场, 密度场等, 表示为

u=u(x,y,z)u = u(x, y, z) u=u(x,y,z)

矢量场

矢量场是物理量为矢量所确定的场, 如力场, 电场等, 表示为

F⃗=F⃗(x,y,z)\vec{F} = \vec{F}(x, y, z) F=F(x,y,z)

方向导数

在函数定义域的内点，对某一方向求导得到的导数。 一般为二元函数和三元函数的方向导数，方向导数可分为沿直线方向和沿曲线方向的方向导数, 表示为

∇vu(l)=∂u∂l\nabla_{v}u(l) = \frac{\partial u}{\partial l} ∇v​u(l)=∂l∂u​

梯度

梯度的方向可以理解为沿变化率上升最快的方向, 一元函数的梯度是标量函数, 多元函数的梯度是向量值函数

三维空间的梯度可以想象为小球在地形上下落的趋势, 小球在无干扰情况下总沿一个方向下落, 这个方向的反方向 ( 即上升的方向 ) 就是梯度的方向
四维空间的梯度可以想象为某个气体分子在冷热不均的空间中总沿最冷方向运动, 这个最冷方向的反方向就是梯度的方向
梯度表示为

grad  u(x,y,z)=ex⃗∂u∂x+ey⃗∂u∂y+ez⃗∂u∂zgrad \; u(x, y, z) = \vec{e_x}\frac{\partial u}{\partial x} + \vec{e_y}\frac{\partial u}{\partial y} + \vec{e_z}\frac{\partial u}{\partial z} gradu(x,y,z)=ex​​∂x∂u​+ey​​∂y∂u​+ez​​∂z∂u​

由哈密顿算符

∇=ex⃗∂∂x+ey⃗∂∂y+ez⃗∂∂z\nabla = \vec{e_x}\frac{\partial}{\partial x} + \vec{e_y}\frac{\partial}{\partial y} + \vec{e_z}\frac{\partial}{\partial z} ∇=ex​​∂x∂​+ey​​∂y∂​+ez​​∂z∂​

则梯度可以表示为

grad  u(x,y,z)=∇ugrad \; u(x, y, z) = \nabla u gradu(x,y,z)=∇u

另外, 标量场的梯度是矢量场

ψ=∫sF⃗⋅dS⃗=∫sF⃗⋅en⃗dS\psi = \int_s \vec{F} \cdot {\rm d}\vec{S} = \int_s \vec{F} \cdot \vec{e_n} {\rm d}S ψ=∫s​F⋅dS=∫s​F⋅en​​dS

en⃗=∇S\vec{e_n} = \nabla S en​​=∇S

散度

散度可以抽象的理解为旋涡向中心收拢 (发散) 的程度
散度表示为

divF⃗=∂F⃗∂x+∂F⃗∂y+∂F⃗∂z{\rm div} \vec{F} = \frac{\partial \vec{F}}{\partial x} + \frac{\partial \vec{F}}{\partial y} + \frac{\partial \vec{F}}{\partial z} divF=∂x∂F​+∂y∂F​+∂z∂F​

由哈密顿算符

∇=ex⃗∂∂x+ey⃗∂∂y+ez⃗∂∂z\nabla = \vec{e_x}\frac{\partial}{\partial x} + \vec{e_y}\frac{\partial}{\partial y} + \vec{e_z}\frac{\partial}{\partial z} ∇=ex​​∂x∂​+ey​​∂y∂​+ez​​∂z∂​

则散度可以表示为

divF⃗=∇⋅F⃗{\rm div} \vec{F} = \nabla \cdot \vec{F} divF=∇⋅F

另外, 矢量场的散度是标量场

旋度

旋度可以理解为三维向量场对某一点附近的微元造成的旋转程度
旋度表示为

rotF⃗=n⃗lim⁡ΔS→0∮cF⃗⋅dl⃗ΔS∣max{\rm rot} \vec{F} = \vec{n} \lim_{\Delta S \to 0} \frac{\oint_c \vec{F} \cdot {\rm d} \vec{l}}{\Delta S} \mid _{max} rotF=nΔS→0lim​ΔS∮c​F⋅dl​∣max​

由哈密顿算符

∇=ex⃗∂∂x+ey⃗∂∂y+ez⃗∂∂z\nabla = \vec{e_x}\frac{\partial}{\partial x} + \vec{e_y}\frac{\partial}{\partial y} + \vec{e_z}\frac{\partial}{\partial z} ∇=ex​​∂x∂​+ey​​∂y∂​+ez​​∂z∂​

则旋度表示为

rotF⃗=∇×F⃗{\rm rot} \vec{F} = \nabla \times \vec{F} rotF=∇×F
